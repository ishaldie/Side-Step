# Side Step - Character Rebrand Analysis

## Current Concept: "Running Shoe"

**The Problem:** The player character is literally a shoe. While unique, this creates several challenges:

| Issue | Impact |
|-------|--------|
| **No face/personality** | Hard to create emotional connection |
| **Limited animation potential** | Shoes don't have expressive features |
| **Difficult to market** | "Play as a shoe" isn't compelling |
| **Art asset complexity** | Need 4 unique shoe designs that read well at small sizes |
| **Upgrade confusion** | "Buy better shoes" when you ARE a shoe is conceptually weird |

## üèÜ TOP RECOMMENDATION: Runner Kid

**Why this works best:**

1. **Preserves your game design** - The shoe upgrade system becomes "kid buys better shoes" which is intuitive

2. **Asset availability** - Kenney's Platformer Pack Redux has ready-to-use character sprites with animations

3. **Marketing appeal** - "Help [Name] run through 5 worlds!" is more compelling than "Play as a shoe"

4. **Customization potential** - Add character skins, outfits, hair styles as additional unlockables

5. **Emotional connection** - Players root for characters with faces

**Suggested Implementation:**

```
Character: "Dash" (or player-named)
- A young runner/skater kid
- Gender-neutral or selectable
- Expressive animations (happy when collecting coins, scared of obstacles)

Progression:
- Barefoot ‚Üí Shows determination, "starting from nothing"
- Flip Flops ‚Üí Beach vibes, casual
- Sneakers ‚Üí Getting serious, athlete mode
- Winged Shoes ‚Üí Mythical hero moment

Skins (future monetization):
- Different hair colors
- Outfit styles (sporty, casual, punk, etc.)
- Seasonal costumes (holiday themed)
```

## Code Impact Assessment

If you choose **Runner Kid**, here's what changes:

| Component | Change Required |
|-----------|-----------------|
| `player.gd` | Replace shoe polygon with AnimatedSprite2D |
| `game_manager.gd` | Rename `SHOES` to `GEAR` or keep as-is (shoes are still upgrades) |
| Shop UI | Update icons, keep shoe purchasing logic |
| Main Menu | Update character display |
| Tutorial | Update "shoe" references to "gear" |

**Estimated effort:** 2-4 hours of code changes + new sprite assets
